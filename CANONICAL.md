# Canonical Repository

The canonical source and release authority for KiCad Studio Kit is:

https://github.com/oaslananka/kicad-studio-kit

All issues, pull requests, CI status, release automation, package publishing,
security advisories, and documentation updates must originate from that
repository.

Known alternate repositories: none under this repository's control.

If an alternate read-only copy is created later, its README must point back to
this canonical repository and must not define independent issues, releases,
publishing workflows, security policy, or automation credentials.
